1. 图及其相关描述¶

1.1. 介绍¶

图：表示一组对象之间的成对关系。

学习图，首先要知道两个定义：

  • 顶点 （也叫结点）

  • 边 （也叫弧）

图一般分为两类：

  • 有向图：拥有有序的结点对

  • 无向图：拥有无序的结点对

1.2. 图的表示¶

通常使用两种方式表示图的数据结构：

  • 邻接矩阵

  • 邻接表

1.2.1. 邻接矩阵¶

将有 V 个结点的图放入 VxV 的 0-1 矩阵中。Aij =1 表示结点 i 与结点 j 相连。

对角线都为 0 表示该邻接矩阵所对应的图不包含任何自循环。这个邻接矩阵只有在表示无向图时才是正确的。

1.2.2. 邻接表¶

邻接表使用数组表示。数组的长度为图中结点的个数。数组的元素为一个单链表，链表内的结点表示与当前下标相连通的结点。
